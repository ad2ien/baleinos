# Baleinos admin app

Manage and edit firefox policies

## Dev

- python Qt application
- form.ui edited with [QtCreator](https://doc.qt.io)

### Start

```bash
# init env
python -m venv ./venv
source ./venv/bin/activate
pip install -r requirements.txt

# start project
python baleinosadminmainwindow.py
```
